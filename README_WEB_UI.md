# SmartTerm - 智能终端增强工具 Web UI

SmartTerm 是一个功能强大的终端增强工具，提供自然语言命令转换、智能提示、远程连接管理等多种功能。该项目现在已整合到现代化Web界面中，可通过浏览器访问。

## 功能特性

### 1. 自然语言命令转换
- 将自然语言指令转换为相应的系统命令
- 智能理解用户意图，生成准确的命令
- 支持基于设备类别的专业命令

### 2. 远程连接管理
- **SSH 客户端**: 支持密码认证和密钥认证
- **Telnet 客户端**: 支持传统Telnet连接
- **连接配置持久化**: 保存连接配置到SQLite数据库
- **多连接管理**: 同时管理多个远程连接
- **安全检查**: 自动检测连接配置的安全问题

### 3. 智能交互
- **AI 模式**: 通过AI理解自然语言执行命令
- **Web 终端**: 基于xterm.js的终端仿真
- **智能补全**: 提供命令自动补全功能
- **命令历史**: 记录执行过的命令

### 4. 会话管理
- **多会话支持**: 同时管理多个远程会话
- **会话状态追踪**: 记录会话的连接状态和统计信息
- **命令记录**: 详细记录每个会话执行的命令

### 5. 设备产品类别管理
- **预定义类别**: 内置多种设备产品类别（网关、OLT等）
- **自动分类**: 根据连接配置自动标记设备产品类型
- **分类统计**: 提供各类设备的使用统计数据

## Web UI 架构

### 后端 (FastAPI)
- REST API 端点提供配置管理、会话管理等功能
- WebSocket 用于实时终端连接
- 集成现有 SmartTerm 组件

### 前端 (React)
- 基于 xterm.js 的 Web 终端仿真
- AI 交互界面
- 连接管理和会话管理界面
- 响应式设计

## 安装和运行

### 后端服务
```bash
# 安装后端依赖
pip install fastapi uvicorn websockets

# 启动后端服务
uvicorn web_app:app --host 0.0.0.0 --port 8000
```

### 前端服务
```bash
# 安装前端依赖
cd frontend
npm install

# 启动前端开发服务器
npm start
```

### 一键启动
```bash
# 使用启动脚本
chmod +x start_web_ui.sh
./start_web_ui.sh
```

## 使用方法

### 1. 连接管理
- 在"连接管理"面板中添加新的连接配置
- 配置主机地址、端口、认证方式等信息
- 保存配置后即可快速连接

### 2. 终端使用
- 选择一个连接配置并点击"连接"
- 在终端面板中直接输入命令
- 或使用AI助手，用自然语言描述操作意图

### 3. AI助手
- 在AI面板中输入自然语言指令
- AI会将其转换为相应的系统命令
- 选择是否执行转换后的命令

### 4. 会话管理
- 查看当前活动会话
- 管理会话历史
- 断开连接

## API 端点

- `GET /api/configs` - 获取所有连接配置
- `POST /api/configs` - 添加连接配置
- `DELETE /api/configs/{id}` - 删除连接配置
- `POST /api/connect` - 连接到远程主机
- `POST /api/disconnect` - 断开当前连接
- `POST /api/command` - 执行命令
- `GET /` - API 根端点
- `WS /ws/terminal` - WebSocket 终端连接

## 安全注意事项

1. **密码存储**: 尽管提供了密码加密功能，但仍然建议使用SSH密钥认证
2. **Telnet警告**: Telnet协议以明文传输数据，不推荐在生产环境中使用
3. **主机验证**: 使用SSH时，确保验证主机密钥以防止中间人攻击
4. **网络隔离**: 在安全的网络环境中使用远程连接功能
5. **Web安全**: 在生产环境中限制CORS策略，添加身份验证

## 扩展功能

- **AI集成**: 基于设备类别文档的智能命令转换
- **插件系统**: 支持插件扩展，可根据需要添加新功能
- **命令审计**: 详细的命令执行记录和审计功能
- **批量操作**: 支持对多台服务器执行相同操作
