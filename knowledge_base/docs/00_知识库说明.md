# ZXA10 命令参考手册知识库

## 快速开始

### 这是什么？

这是中兴 ZXA10 C680/C600/C650/C620 光接入设备的命令行知识库，包含 **5211条命令**，来自官方 V1.2.1 版本命令参考手册。

### 如何查询？

**方式1：按场景查询**
- 查看 `docs/SCENARIO_COMMANDS.md` 场景命令对照表
- 覆盖23个常见场景（BGP配置、QoS、VLAN等）

**方式2：按章节查询**
- 查看 `docs/KNOWLEDGE_INDEX.md` 知识总览
- 13个章节的完整目录

**方式3：按关键字查询**
- 查看 `indexes/INDEX_BY_KEYWORD.md` 关键字索引
- 支持按命令名关键字快速定位

**方式4：按字母查询**
- 查看 `indexes/INDEX_BY_NAME.md` 字母排序索引
- A-Z 完整命令列表

### JSON数据查询

如需程序化处理，使用 `commands/` 目录下的JSON文件：

```json
{
  "name": "reload system",
  "syntax": "reload system force [<reload-reason>]",
  "mode": "特权模式",
  "level": 10,
  "description": "系统复位",
  "chapter": "产品管理",
  "section": "基本配置"
}
```

## 推荐查询流程

```

   ↓
2. 选择查询方式
   ├── 需要配置X → 查看SCENARIO_COMMANDS.md

```
