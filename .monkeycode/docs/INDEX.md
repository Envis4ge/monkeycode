## 项目概述

智能终端系统是一个基于 AI 的命令行工具，能够理解用户的自然语言需求，从知识库中筛选合适的命令，并在用户确认后执行。系统支持本地 Ollama 模型和云端模型两种 AI 模型类型，支持系统命令和自定义脚本两种类型的知识库，旨在简化终端操作，提高用户效率。

**目标用户**:
- 需要频繁使用 Linux/Unix 终端的开发者
- 希望通过自然语言快速执行系统命令的用户
- 需要管理复杂命令和自定义脚本的系统管理员

**核心价值**:
- 通过自然语言理解降低终端使用门槛
- 智能筛选推荐合适的命令，减少记忆负担
- 支持自定义脚本扩展，满足个性化需求
- 本地/云端模型灵活切换，适应不同环境

## 快速参考

### 命令

```bash
# 启动交互式终端
python -m learn_nanobot

# 查看帮助
python -m learn_nanobot --help

# 测试模型连接
python -m learn_nanobot --test-model

# 添加命令到知识库
python -m learn_nanobot --add-command <command>
```

### 重要文件

| 文件 | 目的 |
|------|------|
| `requirements.txt` | Python 依赖 |
| `config.yaml` | 系统配置文件 |
| `knowledge_base.json` | 命令知识库 |
| `models/` | AI 模型相关代码 |
| `cli/` | 命令行接口代码 |
| `core/` | 核心业务逻辑 |
