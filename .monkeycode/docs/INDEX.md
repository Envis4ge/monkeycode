# smart_term 文档

smart_term 是一个面向新手用户的智能终端工具，旨在降低命令行使用门槛，通过自然语言交互、智能提示和命令解释等功能，让初学者能够轻松使用命令行。

## 项目概述

smart_term 是一个智能终端增强工具，它将传统命令行界面与AI能力相结合，为新手用户提供友好的命令行使用体验。系统支持自然语言命令转换、智能命令补全、命令解释和提示以及命令历史管理等核心功能。

**目标用户**: 命令行新手用户、不熟悉Linux/Unix命令的初学者

**核心价值**:
- **降低学习门槛**: 通过自然语言理解，用户可以用中文或英文描述需求，系统自动转换为合适的命令
- **实时指导**: 对每个命令提供详细解释和参数说明，帮助用户理解命令的作用
- **智能辅助**: 基于上下文和用户习惯，智能推荐相关命令和参数
- **历史追溯**: 便捷的命令历史管理和搜索功能，快速复用之前执行过的命令

## 技术选型

| 类别 | 选择 | 理由 |
|------|------|------|
| 编程语言 | Python | 生态丰富、AI库支持好、跨平台、易于新手学习和扩展 |
 | 终端框架 | prompt_toolkit / rich | 功能强大的终端UI库，支持丰富的交互和样式 |
| AI服务 | Ollama（本地）及云端API | 灵活支持Ollama本地模型和OpenAI/Anthropic等云端API，兼顾隐私和性能 |
| 知识库 | SQLite / JSON | 存储命令描述、参数说明、使用示例，支持系统命令和自定义脚本 |
| 命令解析 | argparse / click | 标准Python库，提供灵活的命令行参数解析 |
| 数据存储 | SQLite | 轻量级、无需额外部署、适合本地命令历史和知识库存储 |
| 配置管理 | TOML | 简洁易读、支持嵌套结构、Python原生支持 |

## 核心功能

### 自然语言命令转换
- **目的**: 将用户的自然语言描述转换为可执行的命令行命令
- **描述**: 用户可以用中文或英文描述想要完成的任务（如"列出当前目录的所有文件"），系统利用AI模型理解意图并生成相应的命令（如`ls -la`），并支持预览和确认执行

### 命令解释和提示
- **目的**: 为用户执行的每个命令提供详细的解释和说明
- **描述**: 当用户输入或执行命令时，系统实时显示命令的功能说明、参数含义、使用示例以及可能的注意事项，帮助用户理解命令的作用和风险

### 命令历史管理
- **目的**: 记录、搜索和重用历史命令
- **描述**: 系统自动保存命令历史，支持按关键词搜索、按时间筛选、添加注释标签等功能，用户可以快速查找和重用之前的命令，提高工作效率

 ### 知识库管理
- **目的**: 维护结构化的命令知识库，支持系统命令和自定义脚本
- **描述**: 系统维护包含命令名称、描述、参数、使用示例的知识库，支持用户添加自定义脚本，系统根据用户需求从知识库中筛选最合适的命令

### 智能命令补全
- **目的**: 基于上下文和用户意图提供智能命令建议
- **描述**: 在用户输入命令时，系统根据当前目录、历史命令和上下文环境，智能推荐相关的命令和参数选项，减少用户记忆负担

### 模型切换与回退
- **目的**: 支持多个AI模型配置，实现自动切换和回退机制
- **描述**: 系统支持同时配置多个AI模型（Ollama本地模型、OpenAI、Anthropic等），按优先级使用，当前模型失败时自动切换到下一个可用模型，确保服务可用性
